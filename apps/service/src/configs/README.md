# 配置层(`configs`)

- 配置文件名必须为`[服务名].config.ts`
- 配置文件的配置和类型由`index.ts`统一导出
- 部署`.env`文件里的环境变量名看配置文件的定义
- redis数据库分配(优先使用`REDIS_`开头的环境变量)

  | 数据库 | 作用                     | 完成 |
  | ------ | ------------------------ | ---- |
  | 0      | 用于接口节流             | ✔   |
  | 1      | 用于任务队列             | ✔   |
  | 2      | 用于缓存数据库查询的数据 | ✔   |
  | 3      | 用于存储用户信息(jwt等)  | ✔   |
  | 4      | 用于存储验证码           | ✔   |
